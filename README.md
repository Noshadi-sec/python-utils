# python-utils

A lightweight Python library of utility functions for mathematical operations and common calculations.

## Installation

```bash
git clone https://github.com/Noshadi-sec/python-utils.git
cd python-utils
```

## Usage

```python
from app import add

result = add(5, 3)
print(result)  # 8
```

## Features

- Basic arithmetic operations
- More utility functions being added regularly
